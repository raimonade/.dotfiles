# Security Scan Command

Perform security review on recent changes:

Use the security-reviewer agent to check for:
1. Hardcoded secrets, API keys, credentials
2. SQL injection vulnerabilities
3. XSS vulnerabilities
4. Missing input validation
5. Insecure authentication/authorization
6. Sensitive data exposure
7. OWASP Top 10 issues

Focus on changed files or specify files to scan.
Provide actionable recommendations with file:line references.
