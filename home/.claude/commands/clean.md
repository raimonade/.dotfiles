# Clean Command

Clean up the codebase after development:

1. Remove console.log statements (excluding test files)
2. Remove commented-out code blocks
3. Find and remove unused imports/variables
4. Identify and remove dead code (unreachable functions/exports)
5. Remove unnecessary markdown files created during session
6. Check for TODO comments that should be addressed
7. Run typecheck and linter to verify

Show summary before applying changes.
